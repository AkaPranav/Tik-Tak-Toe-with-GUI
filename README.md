# 🕹️ Tic-Tac-Toe Game using Tkinter

A user-friendly **Tic-Tac-Toe** game built using the **Tkinter GUI library** in Python. This classic two-player game lets users take turns as "X" and "O", providing a fun and interactive experience.

---

## 📌 Features

- 🎮 Classic 3x3 Tic-Tac-Toe board
- 👤 Two-player mode (play with a friend)
- 🖱️ Intuitive button-based interface
- 🧠 Auto-detects win and draw conditions
- 🔁 Option to reset and play again
- ✅ Clean and responsive GUI using Tkinter

---

## 🛠️ Technologies Used

- Python 3.x
- Tkinter (Standard GUI library in Python)
